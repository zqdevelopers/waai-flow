# Code of Conduct

This project should be a useful and respectful open-source space.

## Expected Behavior

- Be respectful and constructive.
- Report bugs with enough detail to reproduce them.
- Discuss automation features responsibly.
- Respect privacy, consent, platform terms, and local laws.

## Unacceptable Behavior

- Harassment, abuse, or discriminatory language.
- Requests or contributions designed for spam, stalking, credential theft, scraping, or evading platform protections.
- Publishing secrets, session data, phone numbers, or private user data without permission.

Maintainers may close issues, reject pull requests, or block users who violate these expectations.
